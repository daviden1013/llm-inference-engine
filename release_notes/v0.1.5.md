## Changes
- Added `chat_async_stream` for asynchronous streaming chat completions.
    ```python
    async def chat_async_stream(self, messages:List[Dict[str,str]], 
                                    messages_logger:MessagesLogger=None) -> AsyncGenerator[Dict[str, str], None]:
    ```

- Refactored `chat` method into `chat` and `chat_stream` for better clarity between standard and streaming chat completions.
    Previously interface 

    ```python
    engine.chat(messages, stream=True)
    ```
    is now split into:
    ```pythonpython
    engine.chat(messages)          # for standard chat completions
    engine.chat_stream(messages)   # for streaming chat completions
    ```
