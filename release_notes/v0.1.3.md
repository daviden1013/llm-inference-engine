## Changes
- Added OpenAI tool call support for `OpenAIInferenceEngine` and `OpenAICompatibleInferenceEngine`.

For non-streaming
```python
response = llm.chat(messages=[{"role": "system", "content": "You are a helpful tool-calling assistant. When making tool calls, you must also justify why you are making the tool call."}, 
                              {"role": "user", "content": "Write a Hello World program in Python and run it with your tool."}],
    tools=[
        {
            "type": "function",
            "function": {
                "name": "run_python_code",
                "description": "Run a Python code snippet and return the output.",
                "parameters": {
                    "type": "object",
                    "properties": {
                        "code": {
                            "type": "string",
                            "description": "The Python code to execute.",
                        },
                    },
                    "required": ["code"],
                },
            },
        }
    ],
    verbose=True,
    messages_logger=messages_logger
)
```

```cmd
{'response': '', 'reasoning': '', 'tool_calls': [{'name': 'run_python_code', 'arguments': '{"code":"print(\'Hello World\')"}'}]}
```

for streaming
```python
response = llm.chat(messages=[{"role": "system", "content": "You are a helpful tool-calling assistant. When making tool calls, you must also justify why you are making the tool call."}, 
                              {"role": "user", "content": "Write a Hello World program in Python and run it with your tool."}],
    tools=[
        {
            "type": "function",
            "function": {
                "name": "run_python_code",
                "description": "Run a Python code snippet and return the output.",
                "parameters": {
                    "type": "object",
                    "properties": {
                        "code": {
                            "type": "string",
                            "description": "The Python code to execute.",
                        },
                    },
                    "required": ["code"],
                },
            },
        }
    ],
    stream=True,
    messages_logger=messages_logger
)

for chunk in response:
    print(chunk)
```

```cmd
{'type': 'response', 'data': ''}
{'type': 'tool_calls', 'data': [{'name': 'run_python_code', 'arguments': '{"code":"print(\'Hello World\')"}'}]}
```
